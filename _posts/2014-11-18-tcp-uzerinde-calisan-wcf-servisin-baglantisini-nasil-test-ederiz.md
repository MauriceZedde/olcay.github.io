**TCP üzerinde çalışan WCF servisin bağlantısını nasıl test ederiz?**

`svcutil net.tcp://localhost:9011/YanitsizBirakmayanServis/mex`

Not: svcutil komutlarını Visual Studio Tools altında &#8220;Developer Command Prompt&#8221; ile çalıştırabilirsiniz.
